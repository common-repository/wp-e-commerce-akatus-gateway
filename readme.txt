The WP e-Commerce Akatus Gateway plugin enables integration with WP e-Commerce and the Akatus Payment Gateway (for Brazilian Merchants).

== Description ==

This plugin provides integration with WP e-Commerce and the Akatus Payment Gateway, a Brazilian online payment gateway.

Tested with WP e-Commerce version 3.8.8.5.

Visit [https://www.akatus.com](https://www.akatus.com "Akatus Online Payment Gateway") for more details on using Akatus.

== Installation ==

There are two methods to install the plugin:

**Copy the file to the WordPress plugins directory:**

1. Make a new directory in [SITE_ROOT]/wp-content/plugins called wp-e-commerce-akatus-gateway
2. Copy the file wp-e-commerce-akatus-gateway.php to this new directory.
3. Activate the newly installed plugin in the WordPress Plugin Manager.


**Install the plugin from WordPress:**

1. Search for the WP e-Commerce Akatus plugin from the WordPress Plugin Manager.
2. Install the plugin.
3. Activate the newly installed plugin.

== Configuration ==

1. Visit the Store settings page, and click on the Payments tab.
2. Tick the check box for Akatus then click the Edit link.
3. Enter your Username, configure other setings and click Update.

You should now be able to test the purchases via Akatus.
